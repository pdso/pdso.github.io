Flask学习笔记之Flask-Bootstrap Flask-Moment

Flask-Bootstrap

pip install Flask-Bootstrap 默认安装的是最新版本。也可以在安装的时候自定义版本。

scripts styles

默认的CDN js和css 加载太慢。替换 js和css路径

{% block scripts %}
<script href="{{ url_for('static', filename='js/jquery.min.js') }}"></script>
<script href="{{ url_for('static', filename='js/bootstrap.min.js') }}"></script>
{% endblock %}

{% block styles %}
<link href="{{ url_for('static', filename='css/bootstrap.min.css') }}" rel="stylesheet">
{% endblock %}

Flask-Moment

pip install Flask-Moment。用来本地化日期和时间

替换CDN js路径 设置默认地区

moment是依赖于jquery的


{% block scripts %}
{{ moment.include_moment(local_js='static/js/moment.js') }}

{{ moment.lang("zh-CN")}}
{% endblock %}
